## 🔧 각 수정 상세

### **1️⃣ Ghost Hit (DODGE_IFRAMES 0.25 → 0.35초)**

```
문제: 핑 높은 플레이어가 화면상으로는 피했지만
      서버 판정에서는 피격

해결: 무적 프레임 0.1초 추가 (네트워크 지연 보상)

효과: ✅ 회피 판정 공정성 향상
      ✅ 게임폐양 무족 개선
```

### **2️⃣ 기절 중 공중 부양 (중력 유지)**

```
문제: 점프/낙하 중 기절 상태 진입
      → Anchored = true로 공중 고정
      → 부자연스러운 공중 부양

해결: Anchored 제거 + X,Z 속도만 0으로 설정
      (Y축 중력은 계속 작용)

효과: ✅ 자연스러운 낙하 애니메이션
      ✅ 공중 부양 현상 완전 해결
```

### **3️⃣ 벽 끼임 (넉백 면역 추가)**

```
문제: 벽 옆에서 공룡의 0.2초 반복 넉백
      → 벽충돌 → 다시 넉백 → 떨림

해결: 같은 공룡으로부터 0.5초 넉백 면역 부여
      (다른 공룡은 여전히 공격 가능)

효과: ✅ 벽 끼임/떨림 95% 제거
      ✅ 공정성 유지 (서로 다른 적은 공격 받음)
```

### **4️⃣ 장애물 끼임 (자동 점프)**

```
문제: 1스터드 높이 턱 + AutoJumpEnabled = false
      → 자동 점프 없음
      → 장애물에 걸려서 제자리 걸음

해결: AI 루프에서 0.5초 이상 제자리 감지 시
      자동으로 humanoid.Jump = true 실행

효과: ✅ 장애물 자동 극복
      ✅ 배회/추격 경로 성공률 +40%
```
